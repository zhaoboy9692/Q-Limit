# Q-Limit 📈

**Q-Limit** is a modern, efficient open-source stock market analysis and AI multi-role assisted decision-making system. It integrates real-time market tracking, full-dimensional technical/financial/valuation analysis, and a distinctive **Multi-Camp AI Debate** feature, dedicated to providing investors with an objective and in-depth market analysis tool.

> 🤖 **About this project:** The frontend rendering, backend architecture design, and all API integrations of this project were collaboratively developed and generated by AI under the user's natural language guidance. This is not only a practical stock analysis platform but also a Proof of Concept (PoC) for the implementation of Agentic AI in a complex full-stack financial project.

## ✨ Core Features

- **🌍 Pure Frontend LLM Proxy Access**: Supports native access for chat-only LLMs. Without needing complex Function Calling support, the system automatically performs **Context Injection**. It queries the latest stock trends, technical indicators, and news in the background and feeds them to the AI, instantly giving your chatbot real-world financial analysis capabilities!
- **⚔️ Pioneering AI Long/Short Debate Arena**:
  - 🐂 **Bull Analyst**: Focuses on uncovering various potential positive factors and upward trends.
  - 🐻 **Bear Analyst**: Acts as the Sword of Damocles, responsible for warning about valuation bubbles and uncovering hidden risks.
  - ⚖️ **Judge**: Conducts an objective review based on the arguments of both the bull and bear sides, ultimately providing realistic and actionable trading advice.
- **📊 Full-Dimensional Dashboard**:
  - K-line charts (daily and minute charts combined with technical indicators).
  - Panel integration: ticker tape quotes, valuation analysis, technical pattern monitoring, and trending news.
- **⚙️ Privacy-Friendly Pure Frontend Configuration**: API Keys and Base URLs for major AI models are all saved locally in the user's browser `localStorage`, completely avoiding the risk of server-side data leakage.

## 🚀 Quick Start

The backend of this project is built with lightweight **Python Flask**, with simple dependencies and extremely fast startup.

### 1. Prerequisites

Please ensure your system has installed:
- Python 3.8+
- [MongoDB](https://www.mongodb.com/) (Local service or Cloud instance)

### 2. Get the Code & Install Dependencies

```bash
git clone https://github.com/zhaoboy9692/Q-Limit.git
cd Q-Limit

# It is recommended to use a virtual environment
python3 -m venv venv
source venv/bin/activate  # Windows users use `venv\Scripts\activate`

# Install dependencies
pip install -r requirements.txt
```

### 3. Configure Database & Start

The project uses MongoDB as a local cache (K-lines, API rate limiting, etc.), so ensure MongoDB is running.
You can check the MongoDB settings in `config.py` of the project, which defaults to `mongodb://localhost:27017`.

**Start the backend server:**
```bash
python3 app.py
```
> The server will start at `http://127.0.0.1:5001` .

### 4. Configure AI Models

After the system starts, visit `http://127.0.0.1:5001` in your browser.
1. Click the **"⚙️ Settings"** icon in the top right corner to open the global settings panel.
2. Switch to the **"🤖 AI Model Configuration"** tab.
3. For "Bull", "Bear", and "Judge", respectively fill in your third-party or native model proxy information (e.g., Base URL, API Key, Model name). Supports all custom endpoints compatible with the standard OpenAI `/v1/chat/completions` format.

## 🛠 Technical Architecture

- **Backend Layer**: Flask + Requests + PyMongo. Primarily responsible for receiving frontend data, querying data through various aggregated market APIs, assembling it, and forwarding SSE streaming requests to third-party AIs.
- **Data Layer**: Uses MongoDB for lightweight request rate limiting and caching of recalculated K-line trends.
- **Frontend Layer**: Vanilla JS + CSS variable customization system + ECharts K-line rendering engine. Full Single Page Application (SPA) interactive experience, with a modern UI and elegant built-in Dark Mode.

## 📖 Usage Guide

- **Search & Watchlist**: Type the stock code (e.g., `AAPL`) in the top search box to get it directly. You can also add it to the Watchlist on the left.
- **One-Click Debate**: Click "One-Click Debate" in the right chat panel. At this time, the backend will automatically grasp various financial and technical factors and organize three AIs to debate in turn.
- **Individual Chat**: Click the "🐂 Bull" or "🐻 Bear" avatar below to ask vertical questions related to specific financial indicators or recent events according to their specific stances.
